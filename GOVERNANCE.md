# Governance
This project for the RHTI template specification is governed by the DEbug Trace and Performance Monitoring (DTPM) SIG.

The group can be joined by RISC-V members at: https://lists.riscv.org/g/sig-debug-trace-perf-mon.

Mailing list archives are available at: https://lists.riscv.org/g/sig-debug-trace-perf-mon/messages.
