# Governance
This project for the RHTI specification is governed by the SoC HC with technical guidance from the Debug Trace and Performance Monitoring (DTPM) SIG.

The groups can be joined by RISC-V members at: https://lists.riscv.org/g/soc-infra and
https://lists.riscv.org/g/sig-debug-trace-perf-mon respectively.

Mailing list archives are available at: https://lists.riscv.org/g/soc-infra/messages and https://lists.riscv.org/g/sig-debug-trace-perf-mon/messages respectively.
